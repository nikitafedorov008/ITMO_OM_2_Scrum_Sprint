Архитектура MVP образовательной платформы (кратко)

Компоненты
-----------
- Frontend: Flutter (клиент для мобильных и web).
- Backend: FastAPI (REST API).
- База данных: PostgreSQL (через docker-compose). Сейчас backend демонстрационно использует in-memory-хранилище для быстрого старта.

Соответствие эпикам
--------------------
- Epic 1 (Авторизация): `backend/main.py` содержит эндпоинты `/register` и `/login` — демонстрационная реализация с хешированием паролей.
- Epic 2 (Курсы): `backend/main.py` содержит `/courses` — возвращает список курсов (demo данные).
- Epic 3 (Видео): `frontend/lib/video_player_screen.dart` — экран с видео-плеером (video_player). В web-версии можно интегрировать video.js.
