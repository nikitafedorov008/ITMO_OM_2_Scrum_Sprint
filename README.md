MVP образовательной платформы для симуляции Scrum-спринта

Описание
-------
Это минимальный каркас (boilerplate) проекта для образовательной платформы, созданный для участия в учебной симуляции Scrum-спринта. В проекте реализованы базовые структуры frontend (Flutter) и backend (FastAPI), а также docker-compose для быстрого поднятия сервисов.

Цели
-----
- Быстро показать «живой» репозиторий с начальным API и фронтендом.
- Покрыть задачи из эпиков: авторизация (регистрация/логин), получение списка курсов, видео-плеер.

Структура проекта (коротко)
---------------------------
- `/frontend` — минимальный Flutter-приложение (каркас). Содержит экран с видео-плеером (Epic 3).
- `/backend` — FastAPI-приложение с эндпоинтами регистрации/логина и получения курсов (Epic 1, Epic 2).
- `/docs` — документация и описания архитектуры.
- `docker-compose.yml` — разворачивает PostgreSQL и backend (локально).

Как запустить (локально, пример)
--------------------------------
1) Запуск бэкенда и базы (через Docker Compose):

   docker-compose up --build

2) Перейти к фронтенду: директория `/frontend` содержит минимальный Flutter-каркас. Откройте в IDE или запустите `flutter run`.

Примечания
---------
- Backend пока реализован как лёгкий каркас с in-memory-хранилищем для быстрых демо (чтобы не блокироваться базой). В коде есть комментарии, где подрубить PostgreSQL (Epic blocker был учтён — добавлен docker-compose).
- Код содержит комментарии, связывающие части с эпиками спринта.
